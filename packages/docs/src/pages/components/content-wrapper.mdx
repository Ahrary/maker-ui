## Layout

In order for the content wrapper to properly place the `SideNav`, `Sidebar`, and `Menu` components, you have to feed it your desired layout. The layout prop currently supports the following strings:

- `content` (default)
- `content-sidebar`
- `sidebar-content`
- `content-sidenav`
- `sidenav-content`

This method of setting a layout's content will be replaced in future versions of Elements.
