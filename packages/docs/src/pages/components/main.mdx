## Semantic Element

`<main />`

| Props        |            Type             |                                                                                                         Description |
| ------------ | :-------------------------: | ------------------------------------------------------------------------------------------------------------------: |
| **children** |           _node_            |                                                                                                            required |
| **maxWidth** | _string_, _number_, _array_ | overrides the default `content.maxWidth` value in `options.js`. Can take an array of values for responsive scaling. |
