## Semantic Element

`<nav />`

| Props               |        Type        |                                                                                                                                                                                                    Description |
| ------------------- | :----------------: | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
| **children**        |       _node_       |                                                                                                                                                                                                       required |
| **animation**       |      _string_      |                        overrides the default animation type specified in `options.js` as `header.mobileAnimation`. Elements currently supports _fade_, _fadeInUp_, _fadeInDown_, _slideRight_, and _slideLeft_ |
| **backgroundColor** |      _string_      |                                                                                                                               overrides the default background color specified as `bg_mobilenav` in `theme.js` |
| **defaultClose**    |       _bool_       | set this prop if you would like the default Close icon in the top right corner of your nav menu. For menus that slideRight or slideLeft, you can also close the menu by tapping an area outside of the drawer. |
| **spring**          |      _object_      |                                    lets you add a spring _config_ object for the MobileNav entry/exit animation. See the [React Spring](https://www.react-spring.io/docs/hooks/api) documentation for details. |
| **width**           | _string_, _number_ |                                                                                                                                                     overrides the nav menu's width for drawer-style navigation |
