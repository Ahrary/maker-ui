# Gatsby Theme Elements

[![npm version](https://badge.fury.io/js/gatsby-theme-elements.svg)](https://badge.fury.io/js/gatsby-theme-elements)

Build responsive Gatsby themes with layouts powered by [Theme UI](https://theme-ui.com/).

Elements takes care of accessibility, responsive navigation, and theming so you can focus on creating awesome content or adding new integrations to your Gatsby site.

Elements is a **parent** theme. It is designed for use alongside other themes
and plugins.
