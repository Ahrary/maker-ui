## Semantic Element

`<aside />`

| Props        |        Type        |                                                                                                                         Description |
| ------------ | :----------------: | ----------------------------------------------------------------------------------------------------------------------------------: |
| **children** |       _node_       |                                                                                                                            required |
| **width**    | _number_, _string_ | overrides the default `sidebar.width` value in `options.js`. Must be a string with a unit, or a number that will default to pixels. |
