## Semantic Element

`<section />`

| Props               |            Type             |                                                                                                                                                                      Description |
| ------------------- | :-------------------------: | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
| **children**        |           _node_            |                                                                                                                                                                         required |
| **backgroundColor** |          _string_           |                                                                                                                   overrides the default `bg_sidenav` color value from `theme.js` |
| **border**          |          _string_           |                                                                        shorthand border syntax that overrides the default border left or right. Example value: `"1px solid red"` |
| **boxShadow**       |      _string_, _array_      |                  overrides the default box shadow (null for desktop and `"1px 1px 8px 1px rgba(0, 0, 0, 0.1)"` for mobile ). Can take an array of values for responsive scaling. |
| **spring**          |          _string_           | lets you add a spring _config_ object for the SideNav mobile entry/exit animation. See the [React Spring](https://www.react-spring.io/docs/hooks/api) documentation for details. |
| **width**           | _string_, _number_, _array_ |                                                                 overrides the default `sideNav.width` value in `options.js`. Can take an array of values for responsive scaling. |
