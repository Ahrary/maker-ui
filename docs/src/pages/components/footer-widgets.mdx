| Props        |   Type   |                                                                                                              Description |
| ------------ | :------: | -----------------------------------------------------------------------------------------------------------------------: |
| **children** |  _node_  |                                                                                                                 required |
| **columns**  |  _bool_  | defaults to `true`. Automatically creates a CSS grid layout with columns corresponding to the number of direct children. |
| **gridGap**  | _string_ |         overrides the `footer.gridGap` from `options.js` and is used to control spacing for default columns (see below). |

## Default Columns

By default, the footer widgets container will count the number of its direct children to create a row of columns. This lets you easily add equal columns that use the `footer.gridGap` defined in `options.js`.

You can disable this behavior by setting `columns` to `false`:

```jsx
<Footer>
  <FooterWidgets columns={false}>
    Custom footer widget content
  <FooterWidgets>
</Footer>

```
