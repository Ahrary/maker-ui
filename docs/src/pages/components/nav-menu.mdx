## Semantic Element

`<nav />`

| Props         |       Type        |                                                                                                                                                               Description |
| ------------- | :---------------: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
| **children**  |      _node_       |                                                                                                                                                                  optional |
| **flex**      |     _boolean_     |                               a shortcut for the `flex` CSS property that determines if the NavMenu flexes to take up all of the remaining space in its parent container. |
| **justify**   | _string_, _array_ | a shortcut for the `justifyContent` CSS property that lets you determine how your menu is justified in its container. Can take an array of values for responsive scaling. |
| **menuItems** |      _array_      |                                                                      an array of navigation items that NavMenu uses to create a semantic nav menu. See below for details. |
| **width**     |     _boolean_     |                                               an optional width property that can be used for controlling split menus or adding horizontal scrolling. Defaults to `auto`. |

## menuItems

To use NavMenu's built-in menu formatter, create an array of links with a `label` and a `path` property:

```jsx
const links = [
  { label: "Home", path: "/" },
  { label: "About", path: "/about" },
  { label: "Contact", path: "/contact" },
  { label: "Twitter", path: "https://twitter.com" },
]

const MyLayout = () => (
  <Layout>
    <Header>
      <Logo />
      <NavMenu flex menuItems={links} justify="flex-end" />

      ...
)
```

NavMenu uses the path property to determine if the link is local (beginning with `/` for a Gatsby Link) or if the destination URL is external. For external links, NavMenu will add `target="_blank"` and `rel="noopener noreferrer"` to the anchor tag.

## Styling

You can add extra or nested styles to the NavMenu with the `sx`, `css`, or `style` prop (ThemeUI, Emotion, or React).

If you prefer, you can also create a variant in `theme.js` called `navlink` that will assign your custom styles to the nested links.

## NOTE

The Elements NavMenu does not yet support nested dropdown menus. This feature is coming soon.
