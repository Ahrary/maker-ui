## Button Location

The SideNavToggle comes with default fixed positioning of `bottom: 25` and `right: 25`. If you are using a `TabBar` component, you may want to adjust these values.

To do so, access the style prop and add a style object or use a library like ThemeUI or Emotion:

```jsx
// React style
<SideNavToggle style={{ bottom: 40, right: 25 }} />

// ThemeUI style
<SideNavToggle sx={{ bottom: 40, right: 25 }} />

// Emotion style
<SideNavToggle css={{ bottom: 40, right: 25 }} />
```

## Styling

You can add extra styles with the `sx`, `css`, or `style` prop (ThemeUI, Emotion, or React).
