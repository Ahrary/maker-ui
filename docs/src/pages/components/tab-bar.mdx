## menuItems

To use TabBar's built-in menu formatter, create an array of links with a `label`, `path`, and `icon` property.

```jsx
import { TwitterIcon, HouseIcon, AboutIcon, ContactIcon } from "../assets/icons"

const links = [
  { label: "Home", path: "/", icon: HouseIcon },
  { label: "About", path: "/about", icon: AboutIcon },
  { label: "Contact", path: "/contact", icon: ContactIcon },
  { label: "Twitter", path: "https://twitter.com", icon: TwitterIcon },
]

const MyLayout = () => (
  <Layout>
    ...
    <TabBar menuItems={links} />
  </Layout>
)
```

Like `NavMenu`, TabBar uses the path property to determine if the link is local (beginning with `/` for a Gatsby Link) or if the destination URL is external. For external links, TabBar will add `target="_blank"` and `rel="noopener noreferrer"` to the anchor tag.

### Alternative Labels

If you use the same menu object that you used with the `NavMenu` component, you can also supply an `alt` property to use instead of `label`. This might be useful if you want to shorten the name of the label in your `TabBar` and keep it full length in your header:

```jsx
const links = [
  { label: "Longer Nav Label", alt: "Shorter", path:"/", icon: HouseIcon },
  ...
]
```
