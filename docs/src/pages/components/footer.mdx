## Semantic Element

`<footer />`

| Props               |            Type             |                                                                                                        Description |
| ------------------- | :-------------------------: | -----------------------------------------------------------------------------------------------------------------: |
| **children**        |           _node_            |                                                                                                           required |
| **backgroundColor** |          _string_           |                                                      overrides the default `bg_footer` color value from `theme.js` |
| **border**          |          _string_           |                 shorthand border syntax that overrides the default border bottom. Example value: `"1px solid red"` |
| **maxWidth**        | _string_, _number_, _array_ | overrides the default `footer.maxWidth` value in `options.js`. Can take an array of values for responsive scaling. |

## Styling

You can add extra styles with the `sx`, `css`, or `style` prop (ThemeUI, Emotion, or React).
