## colorOptions

If you are using an inline SVG for your logo (recommended), you can pass the Logo component an additional color object. The `colorOptions` prop allows your logo to adapt to different color modes.

Simply create an object and define styles for each of the modes you specified in `theme.js`. You can use an `id` or `class` selector to target paths or shapes of your SVG:

```jsx
const logoColors = {
  dark: {
    ".circle": { fill: "#afa7fe" },
    ".square": { fill: "#3bcec5" },
  },
  cream: {
    ".circle": { fill: "#43bd56" },
    ".square": { fill: "#4f86ff" },,
  },
}

const MyLayout = () => (
  <Layout>
    <Header>
      <Logo colorOptions={logoColors} />

      ...
)

```

To see this concept in action, try switching the color mode on this page or refresh and watch the logo change.
